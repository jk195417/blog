# Hong Xin's Blog

部落格用 [hexo](https://github.com/hexojs/hexo) 搭建
主題為 [hexo-theme-icarus](https://github.com/ppoffice/hexo-theme-icarus)
部屬於 Github Pages 上

## Usage

啟動 localhost server

```bash
npm install
npm start
```

產生靜態檔案

```bash
npm run generate
```

部屬至 Github Pages

```bash
npm run deploy
```
